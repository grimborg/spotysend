Spotysend sends commands to Spotify on GNU/Linux over the DBUS.

The available commands are:

* Raise 
* Quit 
* Next 
* Previous 
* Pause 
* PlayPause 
* Stop 
* Play 
* Seek( Offset )
* SetPosition( TrackId, Position )
* OpenUri( Uri )
* Seeked( Position )

For example::

    spotysend PlayPause
